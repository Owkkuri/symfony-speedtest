Apache Solr
A little intro
Tired of writing this?
SELECT *
FROM FancyTable
WHERE UberFirstField LIKE "%bobby%"
       OR GeneralSpecificField LIKE "%bobby%"
       OR KHAAAN LIKE "%bobby%"
       OR SeriouslyMoreFields LIKE "%bobby%"
Why not try this :D
$query = $client->createSelect();
$query->setQuery('bobby');
$resultset = $client->select($query);
What is it?
  • Java search engine based on Lucene
  • Query via HTTP API (kinda RESTful)
  • Give it data, it gives you results.
How to get data into Solr:
Numerous ways, two I like.
  • Data Import Handler
  • Post doc from App
DIH
XML file with SQL queries.
Flattening your DB to get docs.
Doesn't bog down App's ORM with 
pre/post save hooks.
